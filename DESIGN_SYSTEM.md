# Southstyle Suki Points - Design System

## Direction

Mobile-first loyalty card interface inspired by green consumer rewards apps. The customer portal should feel clean, bright, and easy to use at the counter or on a phone.

## Palette

- Primary green: `#078b3e`
- Deep green: `#046b31`
- Soft green: `#e8f7ee`
- Cream app background: `#f6f7ef`
- Paper cards: `#ffffff`
- Ink: `#17221b`
- Muted text: `#738078`
- Scan active blue: `#12a8d8`
- Submit orange: `#f6a11a`
- Redeem danger: `#d92353`

## UI Rules

- Customer screens use a max-width mobile shell.
- Header sections use the green hero surface.
- Content cards are white with subtle shadow and small radius.
- Bottom navigation is fixed, white, and has a featured center Scan action.
- Points display always shows decimal-capable peso value because 1 point equals PHP 1.
- QR/RFID screens must never show sensitive customer data in the scanned code.
